# ROS-SLAM-Docker

## Usage

1. Set up NVIDIA hardware support: `./setup_nvidia.sh`

2. Start docker container: `./compose.sh`

3. Enter the container: `./attach.sh` or `Ctrl+Shift+P` `> Dev Cotainers` commands in vscode
